# Acknowledgements

Context Workbook is an independent project, but its framing and many workflow recipes were directly inspired by Thariq Shihipar's writing and examples about HTML artifacts in agent workflows.

Primary references:

- Thariq Shihipar, [Using Claude Code: The Unreasonable Effectiveness of HTML](https://x.com/trq212/status/2052809885763747935).
- Thariq Shihipar, [The unreasonable effectiveness of HTML — examples](https://thariqs.github.io/html-effectiveness/).
- Thariq Shihipar, [ThariqS/html-effectiveness](https://github.com/ThariqS/html-effectiveness).

The idea this project builds on: rich artifacts can communicate agent work better than a wall of Markdown. Context Workbook extends that direction with a manifest-first, sheet/range-based protocol for lower-token agent continuation, durable archive, and multi-format preservation.

This project is not affiliated with, sponsored by, or endorsed by Thariq Shihipar, Anthropic, X, or GitHub.
